# Habit Tracker App

A mobile application for tracking habits with a buddy system where users can motivate each other with cheer messages and GIFs.

## Technologies Used

- **React Native** - Cross-platform mobile framework
- **Expo** - Development and build platform
- **Firebase Authentication** - User authentication
- **Cloud Firestore** - Real-time NoSQL database
- **React Navigation** - Stack & tab navigation
- **Giphy API** - GIF integration
- **Expo Camera & Barcode Scanner** - QR code buddy pairing
- **React Context API** - State management

## Key Techniques

- Component-based architecture
- Firebase real-time data synchronization
- Async/await for asynchronous operations
- Custom hooks for reusable logic
- Nested navigation with conditional rendering

## Features

- User authentication (register/login)
- Habit creation and tracking
- Buddy system with QR code pairing
- View buddy's habit progress
- Send cheer messages with GIFs
- Profile management
